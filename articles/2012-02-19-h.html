node + npm + coffee 導入

$ curl http://npmjs.org/install.sh | sh $ npm search coffee-script $ npm view !$ $ npm install -g !$ $ coffee -v

coffee 練習

<pre class="code lang-txt" data-lang="txt" data-unlink>$ coffee

coffee> console.log "Hello, world" Hello, world undefined

$ coffee -e "console.log 'Hello, World'" Hello, World

$ echo "console.log 'Hello, World'" > hello.coffee $ coffee -c hello $ ls hello.coffee hello.js

$ node hello Hello, World

$ cat hello.js (function() {

console.log("Hello, World!");

}).call(this);

Vim向け

<pre class="code lang-txt" data-lang="txt" data-unlink># シンタックスとか

$ git clone git://github.com/kchmck/vim-coffee-script.git .vim/bundle/vim-coffee-script

<leader> + r でコンパイル結果を表示する設定

$ git clone git://github.com/thinca/vim-quickrun.git .vim/bundle/vim-quickrun $ vim .vimrc $ tail -n 3 .vimrc let g:quickrun_config = {} let g:quickrun_config['coffee'] = {'command' : 'coffee', 'exec' : ['%c -cbp %s']}
